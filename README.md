# 🚀 Space Invaders Game

A classic **Space Invaders-style arcade game** built using Python and Pygame.
Control your spaceship, destroy enemies, and survive as long as possible.

---

## Features

* Smooth player movement and controls
* Shooting mechanics with multiple lasers
* Enemy spawning and increasing difficulty
* Collision detection system
* Score tracking
* Player lives system
* Sound effects and background music
* Explosion animations
* Game over and restart functionality

---

## Tech Stack

* **Language:** Python
* **Library:** Pygame

## Controls

| Key   | Action          |
| ----- | --------------- |
| ← / A | Move Left       |
| → / D | Move Right      |
| Space | Shoot           |
| Enter | Start / Restart |
| Esc   | Exit            |

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/space-invaders.git
cd space-invaders
```

### 2. Install dependencies

```bash
pip install pygame
```

### 3. Run the game

```bash
python main.py
```

## How It Works (Core Concepts)

* Game loop with frame control (FPS)
* Event handling for keyboard input
* Collision detection using rectangles
* Object movement using coordinate updates
* State management (menu, play, game over)
